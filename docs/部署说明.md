# IT工单系统 — 部署包

## 快速部署

```bash
# 1. 克隆项目
git clone https://github.com/vspoptang/it-ticketing.git
cd it-ticketing

# 2. 创建 .env（修改密码）
cp .env.example .env
# 编辑 .env，修改 DB_PASSWORD 和 JWT_SECRET_KEY

# 3. 启动
docker compose up -d

# 4. 验证
curl http://localhost:8000/health
# → {"status":"ok"}
```

## 包含文件

| 文件 | 说明 |
|------|------|
| `Dockerfile` | Python 3.12-slim + Gunicorn/Uvicorn |
| `docker-compose.yml` | PostgreSQL 16 + App 双容器编排 |
| `.env.example` | 环境变量模板（22个配置项） |
| `requirements.txt` | Python 依赖清单（14个包） |

## 架构

```
                    ┌──────────────┐
     :8000 ←─────── │   App容器    │
                    │ Gunicorn+    │
                    │ Uvicorn      │
                    └──────┬───────┘
                           │
                    ┌──────▼───────┐
                    │ PostgreSQL   │
                    │ 16-alpine    │
                    └──────────────┘
```

## 环境变量（必填项）

| 变量 | 说明 |
|------|------|
| `DB_PASSWORD` | PostgreSQL 密码 |
| `JWT_SECRET_KEY` | JWT 签名密钥（≥32字符） |
| `ADMIN_DEFAULT_PASSWORD` | 管理员初始密码 |

## 系统要求

- Docker 24+ & Docker Compose v2
- CPU: 2核+
- 内存: 2GB+
- 磁盘: 10GB+

## 默认账号

| 用户名 | 密码 | 角色 |
|--------|------|------|
| admin | 由 .env 中 ADMIN_DEFAULT_PASSWORD 设定 | 管理员 |
